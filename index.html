CrossBridge

CrossBridge is the open-source version of Adobe FlasCC (formerly the Alchemy project). It provides a complete C/C++ development environment for targeting the Adobe Flash Runtime.

Features:

  • LLVM-GCC 4.2 compiler with Flash backend
  • GDB debugger that debugs your code while running in the Flash Player
  • SWIG backend that auto-generates inter-operability code, allowing you to build Flash libraries (SWCs) that wrap C/C++ libs
  • Multi-threading based on POSIX Threads

Getting involved in development

Check out the Install Guide

Master branch

  1. Create a fork
  2. Check out GitHub Issue Tracker
  3. Report a new issue, properly describing the situation - backed up with logs
  4. Submit a patch for the issue or ask others to do so
  5. Submit pull requests with complete set of new features

Futures branch

The main goal is to support C++11 by upgrading the compiler infrastructure to the latest LLVM tool-chain (3.X) with FreeBSD 9+

  1. Create a fork
  2. Check out TODOs
  3. Check out LLVM Upgrade
  4. Solve issues and submit pull requests
