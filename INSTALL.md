Install
=======

# OSX

## Build a Mac only SDK.

> "make" 

## Build both Mac and Windows (Cross-Compiled) SDK and package them as DMG and ZIP.

> "make; make win; make deliverables" 


# Windows

## Build a Windows only SDK.

### From Cygwin Shell

> "make all_win"

### From Windows Prompt

> "build-win.bat"

# Linux

## Build a Linux only SDK.

> "make"
